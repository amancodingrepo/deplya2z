# Design Tokens

This file documents the core design system tokens used throughout the SupplyOS application.

## Colors (`C` object)

| Token | Hex | Usage |
| :--- | :--- | :--- |
| `primary` | `#3B82F6` | Primary buttons, active tabs, main accents. |
| `primaryDark` | `#1D4ED8` | Darker variant of primary. |
| `primaryBg` | `#EFF6FF` | Light blue background for active states. |
| `amber` | `#F59E0B` | Warning, pending actions, "Pack" button. |
| `amberBg` | `#FFFBEB` | Warning banners, secondary highlights. |
| `green` | `#10B981` | Success, "In Stock", "Present" status. |
| `red` | `#EF4444` | Errors, "Out of Stock", "Absent" status. |
| `purple` | `#8B5CF6` | Secondary accent, "Catalogue", "Leave" status. |
| `s900` | `#0F172A` | Primary text, titles. |
| `s800` | `#1E293B` | Secondary text, snackbars. |
| `s500` | `#64748B` | Tertiary text, placeholder icons. |
| `s200` | `#E2E8F0` | Borders, dividers. |
| `s50` | `#F8FAFC` | Main application background. |
| `white` | `#FFFFFF` | Card backgrounds, app bar background. |

## Typography
- **Primary Font**: `Sora, sans-serif` (used for headers and logos).
- **Secondary Font**: `DM Sans, sans-serif` (used for body text and labels).
- **Monospace Font**: `JetBrains Mono, monospace` (used for IDs, SKUs, and times).

## Elevation / Shadow
- **Standard Card Shadow**: `0 1px 3px rgba(0,0,0,0.07), 0 1px 2px rgba(0,0,0,0.04)`
- **Phone Shell Shadow**: `0 32px 80px rgba(0,0,0,0.5)`
- **BottomSheet Shadow**: `0 -4px 12px rgba(0,0,0,0.07)`
